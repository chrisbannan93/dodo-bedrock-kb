# Logo header

## Purpose
Use this module to anchor the email with the official Dodo logo. Keep the logo centred and avoid resizing beyond the provided width.

## Content rules
- Use the official logo URL: https://image.email.dodo.com/lib/fe2711717d64047d7d1d79/m/1/4d591d48-bf10-4af5-ae8c-d4a2d33abada.jpg
- Keep padding consistent (20px top/bottom, 30px left/right).
- Maintain a clean white background unless the layout calls for a branded header.
- Do not add additional text inside this module.
